# Flash Car

**Flash Car** — аркадная 2D-игра на **Python** и **Pygame**, в которой игрок управляет автомобилем, уклоняется от препятствий и старается набрать как можно больше очков.

---

## О проекте

Игровой процесс построен вокруг простой, но хорошо масштабируемой механики:
- игрок управляет машиной и перемещается между полосами;
- на дороге появляются препятствия;
- скорость постепенно растёт;
- игра заканчивается при столкновении.

Несмотря на небольшой объём, проект покрывает несколько важных для 2D-игры тем:
- обработку событий клавиатуры и мыши;
- работу с игровым циклом;
- использование `pygame.sprite.Sprite` и групп спрайтов;
- столкновения объектов;
- базовый UI для меню и экрана завершения игры;
- загрузку изображений, шрифтов и музыки;
- адаптацию интерфейса под разные размеры окон.

```text
Flash_car/
├─ main.py

│  ├─ game_state.py
│  └─ game_over_state.py

│  ├─ car.py

   ├─ ui.py

   └─ save_manager.py
```

### Назначение основных модулей

- `main.py` — точка входа и главный цикл приложения.
- `settings.py` — игровые константы и настройки.
- `assets.py` — загрузка изображений, шрифтов и музыки.
- `states/` — состояния игры: меню, игровой экран, экран завершения.
- `sprites/` — игровые объекты: машина, препятствия, разметка дороги.
- `utils/` — вспомогательные функции для UI, layout и сохранений.
- `data/` — ресурсы проекта: изображения, шрифты, музыка.

---

## Архитектура

Проект использует упрощённый подход, близкий к паттерну **State**:
- `StartMenuState` отвечает за стартовое меню;
- `GameState` — за основной игровой процесс;
- `GameOverState` — за экран завершения игры.

Переключение между экранами централизовано через `StateFactory`.

## Управление

### В меню
- запуск игры — кнопка **START**;
- выход из приложения — кнопка **QUIT**;
- ввод cheat-кодов — с клавиатуры прямо в меню.

### Во время игры
- **Left / A** — перемещение влево;
- **Right / D** — перемещение вправо.

### На экране окончания игры
- **R** — начать заново;
- **M** — вернуться в меню;
- также доступно управление мышью через кнопки.

## Cheat-коды

В меню можно вводить специальные коды для смены режима/скина:

- `green`
- `red`
- `black`
- `nyancat`
